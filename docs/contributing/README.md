## 开发环境

确保你安装了 `Python 3.8+`，并执行以下命令来安装 `qq-botpy`与`requests`包：

### 软件包

在开发环境中，我们使用了以下软件包：

- botpy
- requests

您可以执行以下指令安装：

```shell
pip install qq-botpy
pip install requests
```

## 编译

`Modernia` 使用 `PyInstaller` 进行各版本编译。

### 安装

```shell
pip install pyinstaller
```
